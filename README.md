# Welcome Program

A simple Python program that asks the user for their name, age, and favorite hobby, then displays a friendly welcome message.

## Features

* Collects user information using `input()`
* Stores responses in variables
* Displays a personalized welcome message
* Beginner-friendly Python project

## Requirements

* Python 3.x

## How to Run

1. Save the Python code in a file named `personal_intro.py`.
2. Open a terminal or command prompt.
3. Run the program:

```bash
python main.py
```

## Example

### Input

```text
What is your name? Anuj
How old are you? 22
What is your favorite hobby? Travelling
```

### Output

```text
 Welcome! 
Hello, Anuj!
You are 22 years old and your favorite hobby is Travelling.
It's great to meet you! We hope you have a wonderful day and enjoy your hobby!
```
